python programming - Installing a python IDE

19 Mar 2019 - Mark Edosa

Python comes with an integrated development environment (IDE) called IDLE, external or third-party IDEs are also available to enhance development of python applications.

Here are a few of them

  • IDLE

This is python’s default IDE. It requires no installation as it comes with python.

  • Jetbrain’s Pycharm

  • Eclipse via the pyDev plugin

  • Visual studio code

  • Atom

Here’s why you should use a third-party IDE.

  • Syntax highlighting

  • Code Completion

  • Warnings
